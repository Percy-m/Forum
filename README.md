# Forum

企业论坛系统

项目结构：
```
|—— src/main
    |—— java/com/enterprise/forum
        |—— config 应用配置
        |—— constant 常量配置
        |—— controller 控制器
        |—— domain 数据库实体
        |—— dto 数据传输对象
        |—— exception 自定义异常类
        |—— mapper 数据库访问层
        |—— service 服务层
        |—— utils 工具类
        |—— vo 视图对象
    |—— resources
        |—— static 静态文件
        data.sql
        schema.sql
```
